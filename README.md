#Dostaffkin

> Лаконичный SPA на Angular для оформления и трекинга доставок

## О проекте
- Приложение обеспечивает создание/оформление заказов и отслеживание их статуса для службы доставки.
- Интерфейс разбит на страницы: главная, оформление заказа и отслеживание.

## Ключевой функционал
- Главная страница - обзор/стартовая точка.
- Оформление заказа - логика и конфигурация в `src/app/pages/order`.
- Отслеживание статуса доставки - `src/app/pages/track`.
- HTTP-вызовы к бэкенду реализованы в `src/app/services/delivery-api.ts`.
- Преднастроенная папка `docs/` содержит предварительно сгенерированные статические ресурсы.

## Технологии
- Язык: TypeScript
- Фреймворк: Angular 21
- Реактивность: RxJS
- UI/уведомления: `ngx-toastr`
- Инструменты: `@angular/cli`, `vitest` (dev), `jsdom` (dev)

## Быстрый старт
Требования: Node.js и npm (использовано `npm@10.x`).

Установить зависимости:
```

npm install
```

Запустить режим разработки:
```

npm start
```

Сборка для продакшена:
```

npm run build
```

Автосборка (watch):
```

npm run watch
```

Тесты
```

npm test
```

## Структура (важное)
- `src/app/pages/` - страницы приложения (`home`, `order`, `track`).
- `src/app/services/delivery-api.ts` - точка интеграции с API доставки.
- `docs/` - prerendered статические файлы для деплоя.

##
- Backend-эндпоинты и переменные окружения задаются в сервисах/конфигурации (см. `delivery-api.ts`).
- Проект минималистичен: документация и тесты в `vitest` - по необходимости расширить.
